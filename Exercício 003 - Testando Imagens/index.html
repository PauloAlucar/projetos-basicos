Testando o uso de Imagens

Por enquanto, apenas a imagem na tag <img>.

Essa tag pode ter, basicamente, dois atributos: "src", de search, usado para indicar onde está a imagem (na pasta do html ou em um link online) e "alt", de alternative, que é um recurso usado quando não pode ser renderizada a imagem ou para descrever a imagem para portadores de deficiência viual.


Antes da tag Title, insira a linha <link rel="shortcut icon" href="NOME-DA-IMAGEM.ico" type="image/x-icon">

Por enquanto é isso que dá pra fazer sem CSS nem JS.
