Formatação Correta de Textos

Negrito

Há duas maneira de colocar em Negrito, com bold <b>, mas não há informação semântica envolvida; a outra maneira é com <strong>, que possui informação semântica. Recomenda-se o uso de <strong>.

Itálico

Usar a tag <i> faz a alteração sem informação semântica; preferível usar <EM>.

Texto Marcado

A tag <mark> serve para colocar uma marcação como se fosse uma caneta marca-texto. A cor da marcação pode ser alterada por meio de uma alteração de estilo com 'style="', mas pode ser feito um estilo em CSS para alterar todos os mark's.

Texto Normal e Texto Pequeno

Para o HTML, ainda há diferença entre um texto normal (que pode ser aumentdo no CSS) e o Texto Pequeno, que é cercado pela tag <small>. Para casos de apenas ser uma questão de estilo, as alterações devem ser feitas via CSS.

Texto Deletado e Texto Inserido

Para indicar um texto "riscado", a palavra ou termo deve estar cercado de <del>; para indicar que uma lacuna foi preenchida, como em um formulário, a palavra ou termo deve estar cercado de <ins>.

Sobrescrito e Subscrito

Para Sobrescrito, <sup>,e para Subscrito, <sub>

Outras Formatações

Pode-se fazer uma formatação como a que aparece no código-fonte, usando <pre><code></code></pre>, sem a tag de parágrafo.

Exemplo de Texto formatado como Código:

        
    num = int(input('Digite um número'))
    if num % 2 == 0;
        print(f'O número {num} é par')
    else:
        print(f'O número {num} é ímpar')
    print('Fim do programa')

Citações

Usar aspas não é semanticamente uma citação, para tal, deve-se usar a tag <q>, por exemplo: Paulo disse que não está sendo tão difícil quanto parecia.

Para citações longas, o ideal seia usar <blockquote cite='LINK DA CITAÇÃO PARA FACILITAR A LEITURA DOS BUSCADORES'>, exemplo:

A sinonímia é um fenômeno linguístico característico da Semântica e ocorre quando duas ou mais palavras possuem significados semelhantes ou idênticos. Essas palavras são conhecidas como sinônimos. A sinonímia desempenha um papel crucial na comunicação, pois permite a variação lexical, enriquecendo o discurso e evitando repetições desnecessárias.
